# Book-Basket-Backened-NodeJS


#### Email Validation regex

```
^[a-zA-Z0-9.!#$%&'*+/=?^_`{|}~-]+@[a-zA-Z0-9](?:[a-zA-Z0-9-]{0,61}[a-zA-Z0-9])?(?:\.[a-zA-Z0-9](?:[a-zA-Z0-9-]{0,61}[a-zA-Z0-9])?)*$
```

#### Auth routes

```
POST /api/v1/register
POST /api/v1/login
GET /api/v1/auth/:id
PUT /api/v1/auth/:id
DELETE /api/v1/auth/:id
```

#### Book routes

```
GET /api/v1/books
POST /api/v1/books
GET /api/v1/books/:id
PUT /api/v1/books/:id
DELETE /api/v1/books/:id
```
